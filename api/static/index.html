Return current blocking pause state.

      
Result fieldTypeDescription

        
pausedboolWhether blocking is currently paused

api.PauseBlocking

      
Pause all block rules for a specified duration. Blocking resumes automatically when the timer expires.

      
ParamTypeDescription

        
secondsnumberDuration to pause (minimum 1; defaults to 300 if ≤ 0)

      

      
Result fieldTypeDescription

        
pausedboolAlways true after a successful call

        
remaining_secondsnumberSeconds remaining

      

    

    

      
api.ResumeBlocking

      
Resume blocking immediately, cancelling any active pause.

      
Result fieldTypeDescription

        
pausedboolAlways false after a successful call

        
remaining_secondsnumberAlways 0
